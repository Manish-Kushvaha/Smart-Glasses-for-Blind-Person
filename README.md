# Smart-Glasses-for-Blind-Person

👓 Smart Vision – Assistive Glasses for the Visually Impaired

Smart Vision is a wearable assistive system built to help visually impaired users navigate safely through real-time obstacle detection and intelligent voice alerts. The project focuses on embedded system integration, low-latency processing, and seamless hardware–mobile communication.

✨ Project Overview

Built a wearable smart glasses system using ESP32-CAM for real-time obstacle detection

Developed an Android application using Java for Bluetooth communication and audio feedback

Implemented edge-based object detection for faster and low-latency processing

Integrated Bluetooth Serial Communication (SPP) for reliable data transfer between hardware and mobile app

Enabled Text-to-Speech (TTS) for voice-based obstacle alerts

Designed the system to be low-cost, portable, and socially impactful

🧰 Tech Stack

Hardware: ESP32-CAM

Embedded Development: Arduino IDE (Embedded C)

Mobile Development: Android Studio (Java)

Communication: Bluetooth (Serial Port Profile - SPP)

Audio Feedback: Text-to-Speech (TTS) API

🚀 Highlights

Real-time obstacle detection

Edge processing for fast response

Seamless hardware–mobile integration

Reliable Bluetooth-based communication

Practical assistive technology solution
