# 👓 Smart Glasses – for the Visually Impaired

**Smart Vision** is a wearable assistive system built to enable **real-time obstacle detection** and provide **intelligent voice alerts** for visually impaired users. The project focuses on **embedded system integration, low-latency processing, and seamless hardware–mobile communication**.

---

## ✨ Project Overview

- Built a **wearable smart glasses system** using **ESP32-CAM** for real-time obstacle detection  
- Developed an **Android application using Java** for Bluetooth communication and audio feedback  
- Implemented **edge-based object detection** for faster and low-latency processing  
- Integrated **Bluetooth Serial Communication (SPP)** for reliable hardware–mobile data transfer  
- Enabled **Text-to-Speech (TTS)** for voice-based obstacle alerts

## 🧰 Tech Stack

- **Hardware:** ESP32-CAM  
- **Embedded Development:** Arduino IDE (Embedded C)  
- **Mobile Development:** Android Studio (Java)  
- **Communication:** Bluetooth (Serial Port Profile - SPP)  
- **Audio Feedback:** Text-to-Speech (TTS) API
